## üìê Typography Hierarchy

### Font Selection Philosophy

**Playfair Display (Headings)**
- Elegant serif typeface
- Premium, editorial quality
- Excellent for luxury branding
- Used by: Vogue, high-end real estate sites

**Montserrat (Display Text)**
- Modern, professional sans-serif
- Excellent readability
- Versatile for UI elements
- Clean and sophisticated

**Inter (Body Text)**
- Optimized for screen reading
- Professional appearance
- Excellent at all sizes
- Industry standard for quality apps

### Type Scale & Spacing

```
H1: 60-72px (Desktop) / 36-40px (Mobile)
H2: 48-60px (Desktop) / 32-36px (Mobile)
H3: 32-40px (Desktop) / 24-28px (Mobile)
Body: 18-20px (Optimal readability)
Small: 14-16px (Metadata, labels)
```

**Letter Spacing:**
- Headlines: -0.02em (tighter, more elegant)
- Display text: 0.05-0.1em (UPPERCASE labels)
- Body: Default (optimal readability)

### The Rule of Thirds & Golden Ratio
- **Not arbitrary**: Based on visual perception research
- Content areas follow natural eye movement

### White Space is Premium Space
**Real luxury = breathing room**

```
Desktop: 5 columns max
Tablet: 2-3 columns
Mobile: Single column (content-first)
```

## üé≠ Animation Philosophy

### Less is More: Micro-interactions

**Professional animations:**
- **Duration**: 300-600ms (feels natural)
- **Easing**: ease-out (confident, purposeful)
- **Purpose**: Guide attention, provide feedback

**What to Animate:**
- ‚úÖ Hover states (subtle scale, color change)
- ‚úÖ Page transitions (fade-in-up)
- ‚úÖ Image reveals (zoom on hover)
- ‚úÖ Button feedback

**What NOT to Animate:**
- ‚ùå Flashy, attention-seeking effects
- ‚ùå Automatic carousels (poor UX)
- ‚ùå Parallax overload
- ‚ùå Loading spinners everywhere

### Animation Timing

```css
Fast: 200ms - Button clicks, toggles
Standard: 300-400ms - Most transitions
Slow: 500-800ms - Page transitions, modals
```

## üèõÔ∏è Component Design Standards

### Cards: The Premium Property Presentation

**Anatomy of a luxury card:**
1. **Large image** (16:10 ratio) - Hero first
2. **Clear typography hierarchy** - Easy scanning
3. **Subtle shadow** - Depth without drama
4. **White space** - Not cramped
5. **Hover state** - Sophisticated lift

**Shadow philosophy:**
```css
Default: 0 10px 40px rgba(0,0,0,0.15)
Hover: 0 20px 60px rgba(0,0,0,0.2)
```
Mimics natural light, creates depth

### Buttons: Clear, Confident Actions

**Primary buttons:**
- Dark navy background
- White text (high contrast)
- Padding: 16px 32px (comfortable click area)
- Hover: Slight shadow increase

**Secondary buttons:**
- Outlined style
- Matches brand color
- Same size as primary (consistency)

### Forms: Professional Data Collection

**Best practices:**
- Single column (easier to complete)
- Clear labels above inputs
- Adequate spacing (24px between fields)
- Visible focus states
- Error messages that help

## üì± Mobile-First Approach

### Why Mobile-First?

**Statistics:**
- 70% of web traffic is mobile
- Mobile users are equally valuable

### Mobile Optimization

```

```
